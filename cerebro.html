cerebro.cdas.scan_as_json(planner, dataset=None, sql=None, max_records=None)[source]¶

Scans data, returning the result in json format.

Parameters:
  • planner (PlannerConnection) – Planner connection handle.
  • dataset (string, optional) – Name of dataset to scan. dataset or sql must be specified.
  • sql (string, optional) – SQL statement for query. dataset or sql must be specified.
  • max_records (int, optional) – Maximum number of records to return. Default is unlimited.
Returns:

Data returned as a list of JSON objects
Return type:

list(obj)

Examples

>>> import cerebro
>>> ctx = cerebro.create_context()
>>> with ctx.connect_planner(host = 'localhost', port = 12050) as planner:
...     data = cerebro.scan_as_json(planner, dataset = 'cerebro_sample.sample')
...     data
[{'record': b'This is a sample test file.'}, {'record': b'It should consist of two lines.'}]
